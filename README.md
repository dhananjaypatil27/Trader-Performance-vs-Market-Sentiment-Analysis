# Trader-Performance-vs-Market-Sentiment-Analysis
This project analyzes the relationship between Bitcoin market sentiment (Fear/Greed) and trader behavior &amp; performance on Hyperliquid. The goal is to identify patterns that can help design data-driven trading strategies.

The objective is to identify how sentiment influences trading decisions such as leverage, trade frequency, and profitability.

The analysis combines sentiment data with historical trading records to generate data-driven insights and actionable strategies for improving trading performance under different market conditions.

Key Steps
- Data cleaning and preprocessing (handling missing values, duplicates, type conversion)
- Merging datasets on daily time level
- Feature engineering (PnL, win rate, leverage, trade count, long/short ratio)
- Exploratory Data Analysis (EDA)
- Behavioral and performance comparison (Fear vs Greed)

Key Insights
- Trader behavior varies significantly across Fear and Greed market conditions
- Higher leverage usage is observed during Greed periods
- Risk-adjusted performance tends to differ across sentiment cycles
- Distinct trader segments exhibit different strategies and outcomes

Outcome
The project provides actionable trading insights and demonstrates how sentiment-driven analysis can support better decision-making and strategy optimization.
